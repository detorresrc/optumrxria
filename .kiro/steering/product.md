# Product Overview

CORE (Centralized Operations & Revenue Engine) is an Optum enterprise web application for managing client operations, billing, and revenue workflows.

## Key Features
- Single Sign-On (SSO) authentication
- Client management with multi-step forms
- Operational unit configuration
- Contact and address management
- Billing and payment processing

## Target Users
Enterprise users within the Optum/UnitedHealth Group ecosystem requiring centralized operations management.
